>从零开始构建的vue-ssr博客。使用的都是都是最新的技术：Koa2、Vue、Vuex、Vue-server-renderer、PWA、Webpack4、Babel7、MongDB等。
通过 sw-precache-webpack-plugin 插件简单支持pwa，可以在安卓下使用 chrome 浏览器体验，现在支持添加到首屏和离线浏览。

## 用法

```bash
# 安装依赖
npm install

# 开发环境
npm run dev

# 打包
npm run build

# 生产环境
npm run start
```
